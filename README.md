## Instalação

Clone o projeto
```bash
git clone https://github.com/RossyFranca/pokedex.git
```

Instale dependência com npm

```bash
  npm install -g http-server
```

Dentro da pasta do projeto para rodar o servidor

```bash
  http-server .
```
